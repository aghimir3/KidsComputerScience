==================== SLIDE 3 ====================
Command: echo
Print text to the screen — or write text into files!
Print to screen
$ echo "Hello World!"
Displays: Hello World!
Write to a file  (>)
$ echo "My name is Alex" > name.txt
Creates name.txt with that text (overwrites if it exists)
Append to a file  (>>)
$ echo "I love Linux" >> name.txt
Adds a new line to the end of name.txt

==================== SLIDE 4 ====================
Understanding > vs >>
Output redirection lets you save text to files instead of just printing to screen.
>  (Overwrite)
Replaces everything in the file with new content.
$ echo "Hello" > file.txt
$ echo "Goodbye" > file.txt
file.txt now contains:

Goodbye
⚠️  "Hello" is gone — overwritten!
>>  (Append)
Adds new content to the end of the file.
$ echo "Hello" > file.txt
$ echo "Goodbye" >> file.txt
file.txt now contains:

Hello
Goodbye
✅  Both lines are kept!

==================== SLIDE 5 ====================
Command: cat
Short for "concatenate" — it displays the contents of a file in the terminal.
View the contents of any text file
$ cat name.txt
Terminal
$ echo "My name is Alex" > name.txt
$ echo "I am learning Linux" >> name.txt
$ cat name.txt
My name is Alex
I am learning Linux
$ _

==================== SLIDE 6 ====================
Command: cp
Copy files and folders — like Ctrl+C / Ctrl+V but in the terminal!
Copy a file
$ cp notes.txt backup.txt
Makes a copy called backup.txt
Copy into a folder
$ cp notes.txt backups/
Copies notes.txt into the backups folder
Copy a folder
$ cp -r photos/ photos_backup/
The -r flag copies the folder and everything inside it

==================== SLIDE 7 ====================
Command: mv
Move files to a different location OR rename them — one command, two superpowers!
Move a file
$ mv photo.txt images/
Moves photo.txt into the images folder. The original is gone from its old location.
Think of it like cutting and pasting!
Rename a file
$ mv old_name.txt new_name.txt
Renames the file. Linux doesn’t have a separate rename command — mv does both!
Same file, new name!
Key difference from cp: cp makes a copy (original stays), mv moves it (original is gone).
