==================== SLIDE 2 ====================
What is Linux?
Linux is a free and open-source operating system.
An operating system (OS) manages your computer’s hardware and lets you run programs — just like Windows or macOS, but anyone can use, modify, and share Linux for free!
What is “Open Source”?
Imagine you bake a cake and share the recipe with all your friends. They can bake the same cake, add chocolate chips, or share their new version. That’s open source — the “recipe” for Linux is available to everyone!
Free to Use
Never pay a cent
Open Source
Anyone can view & improve the code
Secure
Resistant to viruses
Customizable
Change almost everything

==================== SLIDE 3 ====================
The History of Linux
In 1991, a 21-year-old Finnish college student named Linus Torvalds wanted an operating system he could tinker with. So he built his own — and shared it with the world for free!
1991
Linus Torvalds creates Linux as a hobby project in Finland
1993
First distros appear: Debian & Slackware
1998
IBM invests heavily — Linux goes mainstream
2004
Ubuntu makes Linux beginner-friendly
2008
Google releases Android, built on Linux
Today
Powers the internet, supercomputers, phones & space!

==================== SLIDE 6 ====================
Linux Distributions
Because Linux is open source, many groups have created their own versions called distributions ("distros"). Think of it like ice cream — the base is the same, but each flavor is a little different!
Ubuntu
Most popular for beginners. Easy to use and great for learning.
Fedora
Has the latest software. Popular with developers.
Linux Mint
Beginner-friendly with a Windows-like desktop.
Debian
One of the oldest. Super stable and reliable.
Arch Linux
For advanced users who build their system from scratch.

==================== SLIDE 7 ====================
The Terminal
A text-based way to talk to your computer. Instead of clicking icons, you type commands. It looks intimidating at first, but it’s incredibly powerful!
Terminal
$ ls
Desktop  Documents  Downloads
$ mkdir my_project
$ cd my_project
$ touch hello.txt
$ ls
hello.txt
$ _
Speed
Many tasks are faster by typing than clicking
Power
Some things can only be done through the terminal
Automation
Write scripts to automate repetitive tasks
Remote Access
Often the only way to manage servers

==================== SLIDE 8 ====================
Essential Commands
ls
List files and folders
$ ls
cd
Change directory
$ cd Documents
mkdir
Make a new directory
$ mkdir my_folder
touch
Create a new empty file
$ touch hello.txt
rm
Remove (delete) a file
$ rm hello.txt
rmdir
Remove an empty folder
$ rmdir old_folder
