# Internet Investigator - Classwork Instructions

## What is this assignment?

You're an **Internet Investigator**! In this classwork, you'll:

- **Review** what you learned last week (IP addresses, DNS, gateways)
- **Investigate 3 websites** you actually use (YouTube, TikTok, Discord, etc.)
- **Break down URLs** into their parts (protocol, domain, path)
- **Check security** - does the site use HTTPS?
- **Trace the route** your data takes using the `tracert` command
- **Design your own website** idea
- **BONUS:** Use Developer Tools to count network requests

**Total Points:** 115 (+10 bonus)

## Need Help?

### Running the Tracert Command

**Windows:**
1. Press `Windows + R`
2. Type `cmd` and press Enter
3. Type: `tracert youtube.com` (or your website)
4. Wait for it to finish and count the hops!

**Mac:**
1. Press `Cmd + Space`
2. Type `Terminal` and press Enter
3. Type: `traceroute youtube.com` (or your website)

**Can't run tracert?** Use this website instead:
https://www.traceroute-online.com/

### Opening Developer Tools (for Bonus)

**Windows:** Press `F12`
**Mac:** Press `Cmd + Option + I`

Then click the **Network** tab and refresh the page to see all requests!
