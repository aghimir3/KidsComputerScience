## 🆘 Need Help?

### Opening the Command Line

**🪟 Windows:**
1. Press `Windows + R`
2. Type `cmd`
3. Press Enter

**🍎 Mac:**
1. Press `Cmd + Space` (opens Spotlight)
2. Type `Terminal`
3. Press Enter

---

### Running the Trace Command

**🪟 Windows:**
```
tracert google.com
```

**🍎 Mac:**
```
traceroute google.com
```

*(Replace google.com with any website)*

---

### Opening Developer Tools

**🪟 Windows (any browser):** Press `F12`

**🍎 Mac:**
- Chrome/Edge: Press `Cmd + Option + I`
- Safari: First enable it in Safari > Preferences > Advanced > "Show Develop menu", then press `Cmd + Option + I`

---

### Can't run tracert/traceroute?

Some school networks block it. Use this online tool instead:
https://www.traceroute-online.com/
